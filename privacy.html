Privacy Policy

Effective April 2026

Gingify Desktop App

Gingify Desktop runs entirely on your PC. It does not have a backend, does not send analytics, and does not collect crash reports.

The only network request it makes is checking the GitHub API for updates on startup (api.github.com/repos/IshekKhal/gingify/releases/latest). This request sends no personal data — only a standard HTTPS GET from your IP address, which is no different from opening any web page. You can disable auto-update checks in settings.

No personal data, no crash reports, no analytics — ever.

Gingify Browser Extension

All data stays in your browser. Tab URLs are read to identify domains and stored in session storage, which is cleared automatically when the browser closes.

Your per-domain rules sync via Chrome's own sync infrastructure (chrome.storage.sync) if you are signed into Chrome — this goes through Google's servers, not ours. We do not have servers.

We cannot see your browsing data. We do not receive it. We cannot receive it — there is no server to receive it.

What we do not do

No data is ever sold. No advertising. No third-party analytics. No tracking pixels. No fingerprinting. No telemetry of any kind.
