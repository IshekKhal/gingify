Windows suspends its own Store apps when idle. But it does nothing about Win32 apps — the Electron apps, the game clients, the chat tools. They sit in RAM forever. Until now.

How Gingify works

[ Drop demo.gif here before launch ]
1

Monitors every app running on your PC — RAM usage, idle time, CPU activity

2

When RAM gets tight, snoozes idle apps automatically — or do it manually any time

3

Click any app to wake it instantly. It resumes exactly where it left off. Nothing is lost.

Install in 1 minute — no store required

  1. Download gingify-extension.zip from Releases
  2. Unzip to any folder
  3. Open Chrome → chrome://extensions → enable Developer mode
  4. Click "Load unpacked" → select the unzipped folder
